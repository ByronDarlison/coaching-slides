Organizational Function Chart

KFFM

Key Function Flow Map

  • Map the 3-5 key functions from lead to cash
  • Color code green/yellow/red — winning or losing at a glance
  • Label arrows with widgets — countable things, not dollars
  • Four stages: functions, owners, widgets, critical numbers

Functional Accountability Chart

  • For each function: who owns it and what number tells you if it's winning or losing
  • The test: can the owner move the number by changing how they run the function
  • Set green/yellow/red thresholds for each critical number

Profit/X

  • Identify the single denominator — profit per what — that captures how the company makes money
  • The obvious billing unit is often wrong. X emerges from the widgets on the KFFM.
  • Once identified, every function aligns around one shared measure
